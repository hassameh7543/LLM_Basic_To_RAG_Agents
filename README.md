# 🚀 LLM Basic to RAG Agents

This repository contains a complete learning and implementation journey from **Large Language Model (LLM) basics to Retrieval-Augmented Generation (RAG) Agents**.  
The goal of this project is to demonstrate how modern AI systems integrate **LLMs, vector databases, retrieval pipelines, and intelligent agents** to build powerful AI applications.

---

# 📌 Project Overview

Large Language Models are powerful but have limitations such as:

- Lack of real-time knowledge
- Hallucinations
- Limited context window

To solve these problems, **Retrieval-Augmented Generation (RAG)** is used.  
RAG allows LLMs to retrieve external knowledge from documents, databases, or APIs before generating responses.

This project covers:

- LLM fundamentals
- Prompt engineering
- Embeddings
- Vector databases
- Retrieval pipelines
- RAG architecture
- AI Agents
